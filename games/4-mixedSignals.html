General design

Discussed game ideas at the start of the jam. Decided on a shared character between two different players. Brought up some further improvements to the game idea to make it so that players could feel the other player's control over the shared character more. Ideas included: One player aims the gun, the other player shoots; the gun has significant kickback; one player controls character direction, the other controls character speed (which can be negative).
